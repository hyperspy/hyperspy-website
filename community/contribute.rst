Contribute
==========

HyperSpy is a community project maintained for and by its users. There are many ways you can help!

- Help other users on `gitter <https://gitter.im/hyperspy/hyperspy>`_
- report a bug or request a feature on `GitHub <https://github.com/hyperspy/hyperspy/issues>`_
- Help review `pull requests on GitHub <https://github.com/hyperspy/hyperspy/pulls>`_
- or improve the :ref:`documentation and code<dev_guide>`
- Contributing new functionalities or improving existing ones might be more effort than scripting for personal use, as tests and documentation need to be included, but shared development avoids reinventing the wheel and improve sustainability of the code!
